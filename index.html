Практикуем CSS (Cascading Style Sheets)

Вводная часть

Каскадные таблицы стилей - это свод стилевых описаний, тех или иных HTML тегов (далее элементов HTML), которыйможет быть применён:

  • к отдельному тегу - элементу
  • одновременно ко всем идентичным элементам на
всех страницах сайта.

CSS по сути своего рода дополнение к HTML, которое значительно расширяет его возможности.

Как использовать css?

  1. Описать стиль непосредственно в самом документе:

    <html style="color: black; font-size: 10px;"> </html>

  2. Написать стилевое описание для всех идентичных элементов в html документе:

    <style>
    body{background-color: blue}
    h1{front-size: 20px}
    h1, h2{font-family: Verdana}
    </style>

  3. Вынести стилевое описание элементов в отдельный css файл

    Для этого используется :

                            <link> rel="stylesheet" href="style.css" type="text/css">

Приоритеты стилей (от низшего к высшему):

  1. Связаная таблица стилей (link)
  2. Импортируемая таблица стилей (@import)
  3. Селектор тега
  4. Селектор class
  5. Селектор id
  6. Правило определённое в теге

Кто такие селекторы и с чем их едят

               Селектор:{
                    color:black;
                    font-size: 14px;
                }

В качестве селектора можно указать тег, class или id, однако последние два требуют символ до себя: .class | #id

Объявляются они так: <h1 class="exemple" id="exemple"> </h1>

color: black и font-size 14px - это определения. Они состоят из двух частей:

  • Свойство
  • Значение

Пройдёмся по некоторым стилям

  1. Выравнивание текста: text-align

    Принимает следующие значениея: left, right, center and justify (выравниваение по ширине)

  2. Оформление текста: ext-decoration

    • blink - Мигает
    • line-thtough - Перечеркивается
    • overline - Надчёркивается
    • underline - Подчёркивается
    • none - текст без оформления

  3. Отступ первой строки - text-indent: 10px

    Задаёт отступ первой строки в блоке текста с левой стороны

  4. Трансформация текста - text-transform

    Имеет следующие свойства:

    • none - значение по умолчанию
    • capitalize - каждое слово с заглавного символа
    • lowercase - всё в нижнем регистре
    • uppercase - всё в верхнем регистре

  5. Вертикальное выравнивание - vertical-align

    Имеет следующие значения:

    • baseline - базовая линия элемента по базу родителя
    • bottom - элемент по нижней части строки
    • middle - середина элемента по базе родителя и прибавляется 1/2 высоты родителя
    • sub - нижний индекс
    • super - верхний индекс
    • text-bottom - нижняя граница элемента по нижнему краю строки
    • text-top верхняя граница элемента по верхнему краю строки
    • top

  6. Пробелы и перенос строки - white-space

    • normal - обычный вывод текста
    • nowrap - запрет на автоматический перенос строки
    • pre - показывает так как он есть
